# Análisis de Ventas - Challenge Data Science Latam

Este proyecto realiza un análisis de cuatro sucursales de venta minorista utilizando **Python** y la librería **Pandas**. El objetivo es obtener *insights* estratégicos sobre facturación, popularidad de categorías y satisfacción del cliente

---

## 📊 Descripción del Proyecto

El script procesa datos provenientes de 4 archivos CSV remotos y ejecuta un flujo de trabajo dividido en 5 etapas críticas:

1.  **Facturación Total:** Identifica qué tienda genera más ingresos.
2.  **Ventas por Categoría:** Determina las tendencias de consumo por sucursal.
3.  **Calificación y Satisfacción:** Evalúa la experiencia del cliente mediante lógica condicional.
4.  **Rotación de Productos:** Encuentra los productos estrella y los de menor rendimiento.
5.  **Envío Promedio:** Analiza los costos de envío para optimizar la distribución.

---

## 🛠️ Tecnologías y Librerías

* **Python 3.x**
* **Pandas:** Para la manipulación y limpieza de datos.
* **Matplotlib:** Para la generación de gráficos estadísticos.
